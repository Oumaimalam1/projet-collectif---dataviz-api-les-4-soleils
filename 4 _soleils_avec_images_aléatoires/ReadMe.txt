Ajout de plusieurs fonctions 'randomImagesxxxx'

Les fonctions peuvent remplacer le nom de l'image mis dans les conditions d'Oumaima :

Exemple : "url(img/sunrise.jpg)" remplacé par : randomImageSunrise()

Les images sont stocké dans un tableau dans chaque fonction; je n'ai pas encore mis toutes les images;
mais les fichiers sont déjà créer : suffit de mettre les images dedans; si possible avec des noms simples;
et les rajouter dans le tableau.
